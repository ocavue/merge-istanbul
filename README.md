Merge multiple istanbul coverage reports into one.

## Example

```sh
merge-istanbul --out coverage.raw.json coverage1/coverage.raw.json coverage2/coverage.raw.json
```

## Tests
Simply clone the repo, `npm install`, and run `npm test`
